# FitTracker — 健身记录 APP 项目总览

> 鸿蒙 HarmonyOS NEXT 原生应用 | 适合零基础团队上手

---

## 项目信息

| 项目 | 说明 |
|------|------|
| 应用名称 | FitTracker |
| 目标平台 | HarmonyOS NEXT（鸿蒙） |
| 开发语言 | ArkTS |
| UI 框架 | ArkUI（声明式） |
| 架构模式 | MVVM + Repository |
| 数据存储 | RelationalStore（本地）+ 华为 AGC（云同步） |
| 创建日期 | 2026-05-17 |

---

## 核心功能

1. **每日打卡** — 连续打卡天数（Streak）、日历热力图、智能提醒
2. **训练记录** — 动作/组数/重量/时长、历史最佳、训练模板
3. **体重管理** — 每日录入、周/月/年趋势折线图、BMI、目标进度
4. **感受日记** — 文字记录 + 体力评分 + 心情标签 + 照片

## 扩展功能

5. **训练计划向导** — 减脂/增肌预设方案
6. **动作教学视频** — 内置动作库 + GIF/视频演示
7. **营养/饮食记录** — 卡路里 + 三大营养素
